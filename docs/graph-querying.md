# Graph Querying Patterns

This guide documents how agents and integrations should query the ContextClipper code intelligence graph to obtain symbol and dependency information.

## Core Queries

- **`search_symbols(query)`**: Fuzzy search across the codebase.
- **`get_file_symbols(path)`**: Obtain top-level symbols for a given file.
- **`get_affected(files)`**: Discover dependencies and impact maps.
